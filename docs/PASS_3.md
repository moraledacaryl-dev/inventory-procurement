# Pass 3 — Procurement

Implemented scope:

- Supplier directory and supplier-item commercial references
- Purchase requisitions with submitted, approved, and rejected states
- Supplier quotations and requisition-level quotation comparison
- Purchase orders with approval controls
- Partial and complete receiving against purchase-order lines
- Accepted/rejected delivery quantities
- Stock posting only for accepted quantities
- Purchase returns with stock reversal
- Outstanding-quantity and over-receipt validation
- Procurement and receiving user interfaces

Accounting remains responsible for supplier bills, accounts payable, tax treatment, and payment execution.
