# Integration Contracts
- Accounting exchanges valuation/posting references, goods receipts and bill/payment status.
- POS sends completed sales/refunds; Inventory returns availability.
- Staff supplies active employee identity and role/department references.
- Command Center approved needs create or link to purchase requisitions.

Reliability requires idempotency keys, durable outbox/inbox records, retries, dead-letter visibility and external-ID mappings.
